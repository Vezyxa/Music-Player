Отчёт о тестировании приложения "Музыкальный плеер"

Общие сведения

Название приложения: Музыкальный плеер
Версия: 1.0
Дата тестирования: 8 апреля 2025 г.
Ответственное лицо: Эдуард

Цели тестирования:
Убедиться, что приложение корректно сохраняет и загружает плейлисты.
Проверить функциональность управления треками (добавление, удаление, воспроизведение, пауза, остановка).
Проверить стабильность и производительность приложения.

Окружение для тестирования:
Операционная система: Windows 10
.NET Framework: 4.8
Среда разработки: Visual Studio 2019
Компьютер: Intel Core i5, 8GB RAM
Методы тестирования

Модульное тестирование: Проверка отдельных функций программы.
Интеграционное тестирование: Проверка взаимодействия между компонентами.
Регрессионное тестирование: Перепроверка после внесения изменений.

Результаты тестирования:

Сохранение и загрузка плейлистов:
Успех: Плейлисты сохраняются в папке "Музыка" пользователя.
Успех: Треки внутри плейлистов сохраняются корректно.

Функции управления треками:
Успех: Добавление треков работает корректно.
Успех: Функция удаления треков работает корректно.
Успех: Воспроизведение, пауза и остановка треков работают правильно.

Отображение информации:
Неудача: При вводе названия плейлиста в листе отображается путь к файлам. Исправлено: отображается только введённое название.

Стабильность и производительность:
Успех: Приложение стабильно работает без сбоев.
Успех: Производительность удовлетворительна при работе с несколькими плейлистами.
Выводы и рекомендации

Приложение функционально и удовлетворяет основным требованиям.
Необходимо реализовать функцию удаления треков и убедиться, что все баги исправлены.
Рекомендуется провести дополнительные тестирования после внесения исправлений для проверки стабильности.
